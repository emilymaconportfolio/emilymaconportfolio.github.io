Safety Sam

Safety Sam analyzes time series crime data to determine and visualize route safety. The app enables users to walk between work and home with confidence.

Safe Route was developed in Javascript by using the Google Maps API and the San Francisco Police Crime Data. I discovered the SF Open Data site but the JSON data was hard to understand in a meaningful way. I wanted to visualize the relationship between the routes I had available to walk home and my relative safety, so I created Safety Sam.
