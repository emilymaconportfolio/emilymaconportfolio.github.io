Ionic Rebellion

Ionic Rebellion is a typeface I revived based on Ionic No. 5, the first of Griffith's “Legibility Group” for Linotype.

Ghent

Ghent is a cheeky stencil type family with an affinity for abstracted and rounded forms. Ghent Regular, Bold, and Bold Italic are intended for large-scale display editorial and branding applications. Ghent’s companion Book style is intended for text sizes in print or on screen.

Below you'll find the Type Specimen poster for Ghent, as well as variable font Emotional I created as a quirky adaptation based on the typeface.
